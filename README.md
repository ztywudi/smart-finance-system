# 💰 智能财务系统 — 项目介绍

> 一套面向中国多元会计制度的智能化财务记账平台，支持 **12种会计制度 × 5个行业模板**，覆盖企业、政府、非营利组织、基金、农村合作组织全场景。

---

## 一、核心能力

### 🏛️ 会计制度矩阵（12种）

| 类别 | 制度 | 适用对象 |
|:-----|:-----|:---------|
| **企业类** | 企业会计准则 · 小企业会计准则 · 2001版企业会计制度 · **2026版新准则** | 大中小型企业 |
| **政府及非营利** | 政府会计准则 · 民间非营利组织 · 工会会计制度 | 行政/事业单位 |
| **基金类** | 社会保险基金 · 证券投资基金 · 住房公积金 | 专业基金核算 |
| **农村及合作** | 农村集体经济组织 · 农民专业合作社 | 村集体/合作社 |

### 🏗️ 行业模板（5种，持续扩充）

- **房地产开发**：开发成本/开发产品/预收房款等专用科目
- **商品流通**：商品进销差价/发出商品/委托代销等
- **制造业**：直接材料/直接人工/半成品/制造费用明细
- **建筑施工**：工程施工/工程结算/机械作业
- **餐饮酒店**：食材/酒水/餐具消耗等

> 行业模板 = 基于会计制度自动叠加特有科目，建账时即生效。

## 二、功能总览

### 凭证与账簿
- 凭证录入（科目搜索/借贷平衡校验/辅助核算）
- 审核过账三级流程
- 总分类账 · 明细账 · 科目余额表 · 序时账

### 出纳管理
- 现金银行日记账 · 凭证同步 · **银行对账与余额调节表**

### 应收应付
- 客户/供应商管理 · 应收/应付发票 · 收款/付款
- **账龄分析**（6段：未到期~180天+）

### 固定资产
- 资产卡片 · **3种折旧方法**（直线/双倍余额/年数总和）
- **制度自适应折旧规则**（企业次月/政府当月）
- **拆分/合并/变动历史**全程追溯

### 工资管理
- 自定义工资项目 · 员工管理 · 批量计算
- **个税累计预扣法**（3%~45%七级税率）
- CSV导入导出 · 自动生成凭证

### 现金流量
- **21个标准现金流量项目**，凭证录入时自动标注
- **现金流量表自动生成**（经营/投资/筹资）

### 财务报表
- 资产负债表 · 利润表 · 现金流量表 · CSV导出

### 期初管理
- 科目期初余额导入（CSV模板）
- 固定资产期初卡片批量导入
- **期初上线**一键生成建账凭证

### 期末处理
- 损益结转 · 期末结账

### 同步与备份
- **U盘同步**（导出到U盘/从U盘导入/覆盖保护）
- 本地自动备份（保留10份）

## 四、扩展性设计

### 加一个新会计制度 → 3步
1. 继承 `BaseAccountingSystem`，实现 `get_chart_of_accounts()`
2. 在 `system_registry.py` 注册
3. 重启即用

### 加一个新行业模板 → 3步
1. 继承 `BaseIndustryTemplate`，实现 `customize()`
2. 在 `industry_templates/__init__.py` 注册
3. 建账时即可选择

### 加一个新功能模块 → 4步
1. `core/xxx_service.py` — 服务层
2. `ui/pages/xxx.py` — 页面
3. `main_window.py` 注册导航
4. 数据库加表（自动迁移）
